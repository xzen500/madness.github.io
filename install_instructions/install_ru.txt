Установка

1. Загрузить все файлы на сервер где будет располагаться панель.

2. Файл для отклика бота: index.php (название файла можно изменить).

3. Создать базу данных на хостинге.

4. Установить данные для соединения с базой в конфигурационном файле: inc/config.php

5. Установить права CHMOD 666 на файл: inc/config.php

6. Вход в панель управления: http://ваш_сайт/adm/index.php
